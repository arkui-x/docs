# 用户首选项错误码

> **说明：**
>
> 以下仅介绍本模块特有错误码，通用错误码请参考[通用错误码说明文档](errorcode-universal.md)。

## 15500010 删除用户首选项持久化文件失败
**错误信息**

Failed to delete preferences file.

**错误描述**

删除用户首选项持久化文件失败。

**可能原因**

系统错误导致文件删除失败，可能原因如下：
1. 文件名称不正确。
2. 文件路径不正确。

**处理步骤**

1. 检查文件名称是否正确。
2. 检查文件路径是否正确。
